Como Usar Codespaces

Vantagens do GitHub Codespaces

O GitHub Codespaces permite que você configure um ambiente de desenvolvimento completo na nuvem, eliminando a necessidade de configurar sua máquina local. Isso é especialmente útil para projetos complexos ou para quem utiliza diferentes dispositivos.

Criando uma Conta no GitHub

Para começar, acesse o site do GitHub e crie uma conta gratuita. Com uma conta gratuita, você terá acesso a 60 horas de uso gratuito do Codespaces por mês.

Acesso ao GitHub Education

Se você é estudante, recomendamos que se cadastre no GitHub Education. Isso pode oferecer benefícios adicionais, como mais horas gratuitas no Codespaces.

Como usar o GitHub Codespaces

  1. Crie uma conta gratuita no GitHub.
  2. (Opcional, mas recomendado) Cadastre-se no GitHub Education para acesso gratuito ao Codespaces.
  3. A partir dai você poderá acessar os repositorios e carregá-los diretamente via codespaces
