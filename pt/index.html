LabViR

Laboratório Virtual de Robótica

O Laboratório Virtual de Robótica - LabViR - é uma iniciativa do NuRIA - IFSP (Núcleo de Robótica e Inteligência Artificial) em parceria com o DSI/FEM/Unicamp para disponibilizar uma plataforma aberta para o desenvolvimento de estudos e pesquisas em robótica por meio de ferramentas de código aberto baseadas no ROS através e do uso de ambientes virtuais. A platforma disponibiliza uma série de tutoriais, exercícios e projetos que possibilitam aos usuários adquir conhecimentos os conhecimentos necessários de forma gradativa. Há atividades sobre os pré-requisitos necessários para utilização dos ambientes virtuais - Fundamentos do Linux, Docker, Github e outras, e sobre os conceitos fundamentais do ROS, fundamentos de robótica, Manipulação robótica e Navegação autônoma. Também são disponibilizados ambientes virtuais - LabVirtuais que podem ser replicados e complementados para o desenvolvimento de estudos sobre várias diversos sobre Robótica, IA, Machine Learning e diversas outras áreas. Todos os exemplos e ambientes apresentados podem ser baixados e alterados para o desenvolvimento de novos materiais e estudos. Os ambientes apresentados são baseados no ROS 1 (Noetic), Simulador Gazebo e utilização da linguagem Python. Espera-se que esta plataforma seja ampliada ao longo do tempo a partir de novas contribuições que venham a ser incorporadas. Este é o intuito deste projeto.

Cada módulo é composto por diferentes atividades e tutorais, são sugeridos vídeos, arquivos de configuração, e os códigos poderão ser executados no computador local ou remotamente utilizando o ambiente Codespaces do GitHub. .

Para utilização da plataforma é importante que o usuário tenha uma conta habilitada no GitHub para poder executar os códigos e tutoriais. A utilização do Codespaces é opcional, mas é altamente recomendada para facilitar o acesso aos tutoriais e exercícios. O Codespaces é uma ferramenta que permite criar ambientes de desenvolvimento na nuvem, facilitando o acesso a recursos computacionais e evitando a necessidade de instalação local de softwares. Sua utilização é gratuita para usuários do GitHub (limite de 60hs mensais)e oferece uma experiência de desenvolvimento eficiente e colaborativa. Também é utilizada a plataforma The ConstructSim em algumas atividades.
