Como Usar Docker Localmente

1. Criando uma Conta no GitHub

Para começar, acesse o site do GitHub e crie uma conta gratuita. Isso permitirá que você gerencie seus repositórios e contribua com projetos.

2. Fazendo Fork de um Repositório

Após criar sua conta, localize o repositório desejado no GitHub e clique no botão "Fork" no canto superior direito. Isso criará uma cópia do repositório em sua conta, permitindo que você faça alterações sem afetar o repositório original.

3. Clonando o Repositório

Para baixar o repositório para sua máquina local, use o comando abaixo no terminal: 

                
                    git clone https://github.com/seu-usuario/nome-do-repositorio.git

4. Instalando o Docker

No Linux

Execute os seguintes comandos para instalar o Docker no Linux: 

                
                    sudo apt update
                    sudo apt install docker.io
                    sudo systemctl start docker
                    sudo systemctl enable docker

No Windows

Baixe o instalador do Docker Desktop no site oficial do Docker. Siga as instruções do instalador e reinicie o computador, se necessário.

5. Executando o Docker

Após instalar o Docker, você pode verificar se ele está funcionando corretamente com o comando: 

                
                    docker --version

Agora você está pronto para usar o Docker para executar os exercícios do Laboratório Virtual de Robótica.
