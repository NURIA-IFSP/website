Tutorial Docker para Iniciantes

Aprenda o que é Docker e como utilizá-lo passo a passo

O que é Docker?

Docker é uma plataforma que permite empacotar aplicações e suas dependências em containers. Esses containers garantem que o software funcione de maneira consistente em qualquer ambiente, facilitando o desenvolvimento, teste e implantação de aplicações.

Passo a Passo para Usar o Docker

  1. Instalar o Docker: Acesse docker.com e baixe o Docker Desktop para seu sistema operacional.
  2. Verificar a instalação: Após a instalação, abra o terminal e execute: 
    docker --version
  3. Executar seu primeiro container: Teste com o comando: 
    docker run hello-world
    Esse comando baixa uma imagem de teste e executa um container simples.
  4. Listar containers ativos: 
    docker ps
  5. Buscar uma imagem do Docker Hub: Por exemplo, para baixar uma imagem do Ubuntu: 
    docker pull ubuntu
  6. Rodar um container interativo: 
    docker run -it ubuntu bash
    Isso abrirá um terminal dentro do container Ubuntu.
  7. Parar um container: Primeiro encontre o ID com docker ps e depois: 
    docker stop [ID_DO_CONTAINER]
