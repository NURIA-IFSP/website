LabViR

Virtual Robotics Laboratory

The Virtual Robotics Laboratory – LabViR – is an initiative by NuRIA – IFSP (Nucleus of Robotics and Artificial Intelligence) in partnership with DSI/FEM/Unicamp, aimed at providing an open platform for the development of studies and research in robotics through open-source tools based on ROS and the use of virtual environments. The platform offers a series of tutorials, exercises, and projects that allow users to gradually acquire the necessary knowledge. There are activities covering the prerequisites for using the virtual environments – such as Linux Fundamentals, Docker, GitHub, and others – as well as key concepts related to ROS, robotics fundamentals, robotic manipulation, and autonomous navigation. Virtual environments – VirtualLabs – are also provided, which can be replicated and expanded for the development of studies on a wide range of topics in Robotics, AI, Machine Learning, and related areas. All examples and environments can be downloaded and modified for the development of new materials and research. The environments presented are based on ROS 1 (Noetic), the Gazebo simulator, and Python programming. The goal of this project is to continuously expand the platform through new contributions and collaborative development.

Each module includes various activities and tutorials. Suggested videos, configuration files, and source code are provided, and they can be run locally or remotely using GitHub Codespaces.

To use the platform, it is important that the user has an active GitHub account in order to run the provided code and tutorials. While the use of Codespaces is optional, it is highly recommended to facilitate access to tutorials and exercises. Codespaces is a tool that allows the creation of cloud-based development environments, making it easier to access computing resources and eliminating the need for local software installation. It is free for GitHub users (up to 60 hours per month) and offers an efficient and collaborative development experience. Some activities also make use of the The ConstructSim platform.
