Using Docker Locally

1. Creating a GitHub Account

To get started, go to the GitHub website and create a free account. This will allow you to manage your repositories and contribute to projects.

2. Forking a Repository

After creating your account, find the desired repository on GitHub and click the "Fork" button in the top right corner. This will create a copy of the repository in your account, allowing you to make changes without affecting the original.

3. Cloning the Repository

To download the repository to your local machine, run the following command in your terminal: 

                
                    git clone https://github.com/your-username/repository-name.git

4. Installing Docker

On Linux

Run the following commands to install Docker on Linux: 

                
                    sudo apt update
                    sudo apt install docker.io
                    sudo systemctl start docker
                    sudo systemctl enable docker

On Windows

Download the Docker Desktop installer from the official Docker website. Follow the installation instructions and restart your computer if necessary.

5. Running Docker

After installing Docker, you can verify if it is running correctly with the command: 

                
                    docker --version

You are now ready to use Docker to run the exercises from the Virtual Robotics Laboratory.
