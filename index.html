Laboratório Virtual de Robótica

A Robotics Academy é uma coleção de exercícios e desafios de código aberto para aprender robótica de maneira prática. Há exercícios sobre programação de drones, sobre visão computacional, sobre robôs móveis, sobre carros autônomos, etc. É baseado principalmente no simulador de Gazebo e no ROS. Os alunos programam suas soluções em linguagem Python.

Cada exercício é composto de (a) arquivos de configuração do Gazebo, (b) um modelo da web para hospedar o código studentilitis e (c) conteúdos teóricos. Os alunos inserem seu código no arquivo de modelo e usam a API simples fornecida para acessar leituras de sensores e comandos de atuadores (HAL API) e a API simples fornecida para Interface Gráfica do Usuário e depuração (GUI API).

Para execução o aluno lança uma imagem Docker contendo uma simulação Gazebo e se conecta a um servidor web Django para inserir seu código.
